# Overview

Personalization is the core engine of modern user retention. This repository houses two distinct recommendation systems—one for books and one for movies—demonstrating multiple approaches to solving the "cold start" and data sparsity problems.

## The Architecture

* **Content-Based Filtering:** Utilized TF-IDF vectorization and Cosine Similarity to recommend items based on metadata (genres, authors, descriptions).
* **Collaborative Filtering:** Implemented user-item matrix factorization (SVD) to surface recommendations based on aggregated user behavior and rating histories.
