# Overview

Modern supply chains are highly sensitive to demand shocks. This project develops a machine learning solution to forecast product demand across multiple distribution centers, aiming to minimize stockouts while reducing warehouse holding costs.

## The Architecture

* **Feature Engineering:** Constructed features incorporating seasonality, promotional events, and historical lead times.
* **Predictive Modeling:** Trained an `XGBoost` regression model, leveraging its gradient boosting framework to capture complex, non-linear relationships in the supply chain data.
* **Business Impact:** The model outputs dynamic safety stock recommendations, directly translating predictive accuracy into inventory cost reduction.
